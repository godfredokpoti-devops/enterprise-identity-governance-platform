# Zero Trust Security Policy

## Core Principles

- Never Trust
- Always Verify
- Continuous Authentication
- Least Privilege Access

## Security Controls

### MFA

All users must authenticate using MFA.

### Device Trust

Only approved devices may access enterprise systems.

### Network Segmentation

Access is restricted based on user identity and device posture.

### Continuous Monitoring

All authentication events are logged and monitored.

## Compliance Alignment

- NIST
- SOC2
- ISO27001
